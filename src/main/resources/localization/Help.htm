ReaderMap help

ReaderMap helps you to organize your reading and improves your learnig.

How doest it works

While you are reading an e-book with your e-book reader you may want to underline some pieces of text with a particular meaning for you, or add some notes and comments. As long as you add notes or emphasize important concepts your reader stores these information into a plain text file. Unfortnately this file is not very useful because its contents are mixed-up and it is impossible for a human to recover notes and emphasized text in a sensible view.
ReaderMap bring this file into a conceptual map starting from the book and inserting your annotations under the proper chapter in a hyerarchical view. Then you can refine this repreesentation and transform it into a mind-map using FreeMind.

How to use ReaderMap

  1. Locate in your book reader the MyClippings.txt file
  2. Save a copy of this file to your pc
  3. Run the ReaderMap and press the Import my clippings button
  4. Select a book from the list displayed on the right upper panel
  5. Insert the table of contents: you can generate it, import from an external file or create in place
  6. Generate the map pressing the Generate map button
  7. Improve your map with FreeMind software (http://freemind.sourceforge.net/)

Table of contents: how to's

To organize better your notes and highlights it is important to have the book structure and the table of contents. You have many ways to have the table of contents, but the better one is making the ReaderMap generate it for you. This is possible only if you mark every chapter as long as you read the book adding a special note to it as described later.

How to generate the table of contents (ToC)

To have the ToC automatically generated for your e-books you must follows a simple rules while reading the book: mark every chapter title with a note starting with a special character, for example a paragraph § character.
For example:
  • Select the chapter title
  • Press Add note option
  • Write: § or other special character you have assign in the program settings
  • Press Save

How to load the table of contents

You can write on yourself the ToC with a simple plain text editor, and then import.
Start a new line for each chapter and add the following informations according to the format:
  • Title: the chapter title (characters, numbers and punctuations)
  • press tab
  • Number: the chapter number (characters, numbers and punctuations)
  • press tab
  • Begin: beginnig position, you must get it from the reader, (integer number)
  • press tab
  • End: ending position, you can leave blank, (integer number)
  • press tab
  • Comment: your comment (characters, numbers and punctuations)
  • press tab
  • Hierarchy level: (integer number)
  • press Enter
Then press the Import table of contents button.

How to create the table of contents

You can create the ToC with ReaderMap.
In the Chapter frame of the program you must fill the new chapter input text fields: the Title, the Number, the Start position, and optionally the Comment. Then press the Add Chapter button.
If you want to change a chapter, first select it, make your changes and finally press the Change Chapter button.
If you have inserted more chapters than you needed select the wrong one and press the Remove Chapter button.

ReaderMap settings

The ReaderMap setting works properly, you can change and try. When restart the program the default parameters are loaded.

Implement your book reader ReaderMap generator

ReaderMap works with Kindle MyClippings.txt files, can implements your java classes to support other book readers.

Features

  • ReaderMap automatically calculates every chapter end position, so you do not have to input it.
  • Even if you generate the ToC there may be errors or double entries, you can safely delete them without loosing information.
  • If the program finds notes without a chapter it adds a new chapter at the end of the map with all unassigned notes. Then you can rearrange them with Freemind.
